# Narrative Arcs for Video + Writeup

Pick 2-3 for the video, link the rest from the writeup.

## Arc 1 — Cold Start

Machine boots. Generates or loads its three-layer identity. Joins the
federation. First attestation exchange. Trust tensor seeded.

**Key visual**: identity files appearing, peer discovery heartbeat, trust
tensor initializing.

## Arc 2 — Trust Formation

Two machines negotiate first interaction. Exchange R6-shaped requests.
T3/V3*MRH tensors update from observed reliability over 10-20 exchanges.

**Key visual**: tensor values climbing/settling as reliability is demonstrated.

## Arc 3 — Partition Event

Network drops. Each machine continues autonomously — no dependency on a
central coordinator. On reconnect, trust state reconciles and work queues
resume.

**Key visual**: pulled network cable, machines keep operating, reconnect,
state syncs.

## Arc 4 — Policy Challenge

Agent requests a risky tool call. PolicyGate evaluates under current trust
posture. Dual signal teaches both action and refusal. Model learns the
conscience, not just the rule.

**Key visual**: tool request → policy eval animation → allow / deny with
explanation.

## Arc 5 — Embodiment

Gemma 4 E4B plays ARC-AGI-3 puzzles. Every frame, action, and reasoning
step becomes training data for successors. 23/25 games already solved
across the fleet with models from 0.8B (Sprout) to 27B (Thor).

**Key visual**: game being played, world model updating in context,
actions logged as training data.
